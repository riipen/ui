Divider

A divider is a thin line that groups content

Dividers separate content into clear groups.

Variants

The divider emulates <hr> but is rendered as a <div>. The variants available are shown below.

import React from "react";

import Divider from "@riipen-ui/components/Divider";

export default function Variants() {
  const style = {
    height: "10px"
  };

  return (
    <div>
      <Divider variant="fullWidth" />
      <div style={style} />
      <Divider variant="middle" />
    </div>
  );
}
